Usage
-----

Update by path
^^^^^^^^^^^^^^

1. ``<your-domain>/update-by-path/<secret>/<fqdn>``
2. ``<your-domain>/update-by-path/<secret>/<fqdn>/<ip_1>``
3. ``<your-domain>/update-by-path/<secret>/<fqdn>/<ip_1>/<ip_2>``


Update by query
^^^^^^^^^^^^^^^

``<your-domain>/update-by-query?secret=<secret>&fqdn=<fqdn>``
